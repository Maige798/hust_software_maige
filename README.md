## 项目介绍

基于后台的机器翻译引擎，专业翻译人员或翻译用户通过计算机辅助翻译系统方便高效完成翻译工作。

## 项目需求

1. 翻译记忆库管理 
* 检索 
* 浏览 
* 保存
* 导入/导出
* 删除
* 修改
2. 术语库管理
* 创建
* 增添
* 删除
* 匹配
3. 文档操作
* 文档分段
* 文档合并
4. 项目管理
* 新建项目
* 保存项目
* 打开项目
5. 各种格式文件翻译文本导入
6. 翻译编辑
* 手动编辑
* 应用记忆库翻译结果
7. 机器翻译
* MT引擎接口
* Office自动翻译插件
8. 语言识别
9. 拼写检查 
10. 自动完成
11. 导出翻译文件
